# Calculator System Project

This is a C program that reads infix equations from an input file, validates them, converts valid ones to postfix notation, evaluates them, and provides various outputs as specified.

## Files
- `P1_1220022_emam_samara_1.c`: The main C source code.
- `input.txt`: Sample input file with equations.
- `output.txt`: Generated output file with results.

## Compilation
```bash
gcc -std=c11 -Wall -Wextra P1_1220022_emam_samara_1.c -o calculator
```

## Usage
Run the program and follow the menu options:
1. Read equations from file (provide filename, e.g., input.txt)
2. Check validity
3. Convert to postfix
4. Evaluate
5. Print invalid equations
6. Expression tree
7. Save report to output.txt
8. Exit
